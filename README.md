# Todo Application

## Overview
A simple and responsive Todo application built using HTML, CSS, and JavaScript. It allows users to manage daily tasks with persistent storage and interactive UI.

## Features
- Add new tasks
- Mark tasks as completed
- Delete tasks
- Data stored using localStorage
- Completion animation (confetti effect)

## Tech Stack
- HTML
- CSS
- JavaScript (DOM manipulation, events, localStorage)

## How to Run
1. Clone the repository
2. Open `index.html` in browser

## Future Improvements
- Edit tasks
- Filter (All / Completed / Pending)
- Drag and drop sorting
